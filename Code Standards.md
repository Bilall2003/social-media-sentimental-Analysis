## Coding Standards

- **Python version**: 3.10+
- **Follow PEP8**: Use proper indentation, variable names, and spacing.
- **Imports**:
  - Standard libraries first (os, re, time, etc.)
  - Then third-party libraries (numpy, pandas, sklearn, etc.)
  - Then local imports
- **Functions and variables**: Use descriptive names (e.g., `preprocess_text` instead of `pt`)
- **Docstrings**: Use docstrings for all functions and classes
